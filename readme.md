# Ribbbon

Ribbbon is a project management system built on Laravel 5.1.*

# Features
  - User / account creation
  - Client creation
  - Project creation
  - Task creation 
  - Assigning weights to tasks
  - file uploads
  - project sharing

# Get involved

#Installation
-   Clone the repo
-   Run composer install
-   Run php artisan migrate
-   Run php artisan db:seed
-   Start developing!
